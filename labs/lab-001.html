LAB-001 · Active Directory

Implantação de Domínio Active Directory

Criação de um domínio Active Directory do zero, com configuração de DNS, DHCP, unidades organizacionais, usuários, grupos de segurança e permissões de acesso.

Concluído

Objetivo

Simular, em ambiente de laboratório, a implantação de um controlador de domínio corporativo, estabelecendo a base de identidade e autenticação para toda a infraestrutura de rede.

Processo

  • Instalação da função AD DS (Active Directory Domain Services) no Windows Server.
  • Promoção do servidor a controlador de domínio e criação da floresta/domínio.
  • Configuração das zonas de DNS integradas ao Active Directory.
  • Configuração de escopo DHCP para distribuição automática de endereços IP.
  • Criação da estrutura de Unidades Organizacionais (OUs) por departamento.
  • Criação de usuários, grupos de segurança e atribuição de permissões.

Desafios encontrados

O principal desafio foi planejar uma hierarquia de OUs que fosse ao mesmo tempo organizada e escalável, evitando sobreposição de políticas de grupo aplicadas posteriormente. Também foi necessário garantir que a zona de DNS reversa estivesse corretamente configurada para resolução completa de nomes.

Resultado

Domínio funcional com resolução de nomes interna, distribuição automática de IPs via DHCP e uma estrutura organizacional de usuários e grupos pronta para receber políticas de GPO (ver LAB-003).
