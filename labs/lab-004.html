LAB-004 · PowerShell

Automação de Usuários via PowerShell

Script em PowerShell para leitura de um arquivo CSV e criação automatizada e em massa de usuários no Active Directory, reduzindo tarefas manuais e repetitivas.

Planejado

Objetivo

Eliminar o trabalho manual e repetitivo de criação de usuários, permitindo que uma lista de novos colaboradores em CSV seja transformada em contas ativas no Active Directory em segundos.

Processo

  • Importação do módulo ActiveDirectory para PowerShell.
  • Leitura de um arquivo CSV contendo nome, sobrenome, departamento e cargo dos novos usuários.
  • Construção de um script que gera login e senha inicial padronizados.
  • Criação automática do usuário na OU correspondente ao departamento.
  • Registro de log de execução para auditoria do processo.
  • Testes com massa de dados fictícia antes da aplicação em produção.

Desafios encontrados

O maior desafio será tratar exceções — como nomes duplicados ou departamentos inexistentes — sem interromper a execução do script para os demais registros do arquivo.

Resultado

Laboratório planejado como próxima etapa prática, consolidando os conhecimentos de Active Directory (LAB-001) com automação via PowerShell.
